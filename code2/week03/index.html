Javascript-style objects with P5.js

This sketch creates a "Bubble" object with display(), move(), and clicked() functions. The code creates an array to hold them, pushing instances of the object into the array whenever the mouse is clicked.

Note that we can just use a regular old array to have a dynamic size - we don't need to say explicitly how big it is like we do in Processing/Java. We can add objects to the array as we like.

The draw function loops through the array and draws and moves each one. Finally if the array is over 50 positions large, we remove the object in the first position in the array with the "splice()" function.

Javascript-style objects with P5.js part 2

Same basic idea as above, now with a lifespan that affects the transparency of each bubble.

Detecting object-to-object overlap

Bubble objects now detect collision by having each Bubble loop through the entire Bubble array and check for distance. If the distance is less than their combined radii, the fill changes to a random color.

HTML and DOM via P5.js basics

This sketch plays with a bunch of DOM elements. The first one is an h2 header element that is assigned to var h2. That h2 element's position is modified by clicking the mouse- changing both the text with h2.html() and changing the absolute position with h2.position.

This sketch also uses a double-nested loop to create 100 buttons with line breaks splitting up each row, and sets the canvas position to absolute at 0,0. Pressing the mouse button also appends a new paragraph element to the page.

Setting CSS style via P5.js

This sketch first creates a paragraph element, then gives that paragraph a number of style attributes; changing the background color to green, the font family to sans-serif, the font size to 18px, and the color to black.

Next a button is created that when pressed calls the "changeStyle()" function, changing the styling of that text element, via an event handler. When the mouse hovers over that element however, it calls "revertStyle()" and changes back.
