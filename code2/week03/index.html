Javascript-style objects with P5.js

This sketch creates a "Bubble" object with display(), move(), and clicked() functions. The code creates an array to hold them, pushing instances of the object into the array whenever the mouse is clicked. Note that we can just use a regular old array to have a dynamic size - we don't need to say explicitly how big it is like we do in Processing/Java. We can add objects to the array as we like. The draw function loops through the array and draws and moves each one. Finally if the array is over 50 positions large, we remove the object in the first position in the array with the "splice()" function.
